# PLG-4001 Plugin Host 状态转换非法

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

PLG-4001 表示 Plugin Host 状态转换非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

对未发现、非 ready、无 runtime、已 rollback 或其他不允许状态执行 lifecycle/transaction 操作

## 建议操作

读取当前 Host snapshot，并使用该状态允许的命令

- 触发该错误的操作
- 可复现时的最小文档或配置
