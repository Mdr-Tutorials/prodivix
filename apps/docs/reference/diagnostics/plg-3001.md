# PLG-3001 Required capability 被拒绝

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | PLG          |
| 范围     | 插件         |
| 严重程度 | `error`      |
| 阶段     | `permission` |
| 可重试   | 否           |

## 含义

PLG-3001 表示 Required capability 被拒绝。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

effective PermissionSnapshot 中至少一个 required capability 为 deny

## 建议操作

授予所需能力，或保持插件为 blocked/disabled

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
