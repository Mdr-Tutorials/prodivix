# PLG-4033 Gateway response 不符合 contract

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `gateway` |
| 可重试   | 否        |

## 含义

PLG-4033 表示 Gateway response 不符合 contract。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Host handler 返回值未通过 method-specific response Schema 或 byte limit

## 建议操作

修复 Host service adapter 的稳定 projection
