# PLG-4050 Official host implementation 未通过构建证明

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `registry` |
| 可重试   | 否         |

## 含义

PLG-4050 表示 Official host implementation 未通过构建证明。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

package trust、publisher verification、plugin id、package digest 或 package coordinate 与 Web build 中的 official catalog 不一致

## 建议操作

安装与当前 Prodivix build 匹配的官方插件 package，或移除 privileged host implementation 引用

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
