# ANI-5101 CodeSlot 执行能力不可用

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ANI       |
| 范围     | 动画      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

ANI-5101 表示 CodeSlot 执行能力不可用。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

timeline 绑定 custom easing、script 或 shader，但当前 provider 没有对应 execution capability

## 建议操作

选择具备该能力的 runtime，或移除绑定

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
