# PLG-1013 Contribution contract 不受支持

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PLG      |
| 范围     | 插件     |
| 严重程度 | `error`  |
| 阶段     | `schema` |
| 可重试   | 否       |

## 含义

PLG-1013 表示 Contribution contract 不受支持。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Host 未注册 Manifest 声明的 exact contribution point 与 contract version

## 建议操作

使用当前宿主支持的 point/version，或安装提供该 contract 的宿主版本

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
