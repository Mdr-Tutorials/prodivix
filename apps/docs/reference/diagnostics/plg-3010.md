# PLG-3010 Contribution identity 冲突

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `registry` |
| 可重试   | 否         |

## 含义

PLG-3010 表示 Contribution identity 冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

同一 stable `<pluginId>/<contributionId>` 被重复 stage 或与已提交 record 冲突

## 建议操作

为 contribution 使用唯一 local id，并重新执行完整注册事务

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
