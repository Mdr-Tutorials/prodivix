# PLG-2003 当前宿主版本不兼容

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2003 表示 当前宿主版本不兼容。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

当前 Prodivix host version 不满足 `engines.prodivix`

## 建议操作

安装兼容的插件版本，或升级 Prodivix

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
