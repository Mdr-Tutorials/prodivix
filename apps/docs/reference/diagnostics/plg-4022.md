# PLG-4022 Protocol sequence 非单调

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `protocol` |
| 可重试   | 否         |

## 含义

PLG-4022 表示 Protocol sequence 非单调。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

对端 sequence 重复、回退、跳号或超出安全整数上限

## 建议操作

以全新 session 从 sequence 1 重启 runtime
