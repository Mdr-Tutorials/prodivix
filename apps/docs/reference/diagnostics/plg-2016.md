# PLG-2016 UI entrypoint id 重复

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2016 表示 UI entrypoint id 重复。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`entrypoints.ui` 中多个入口使用同一个 local id

## 建议操作

为每个隔离 UI surface 分配唯一 id

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
