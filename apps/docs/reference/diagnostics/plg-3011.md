# PLG-3011 Registry transaction revision 冲突

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `registry` |
| 可重试   | 是         |

## 含义

PLG-3011 表示 Registry transaction revision 冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

commit 时 registry revision 或 permission revision 已不同于 transaction 捕获值

## 建议操作

读取最新 Host snapshot，并重新 prepare 和提交完整 transaction

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
