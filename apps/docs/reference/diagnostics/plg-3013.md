# PLG-3013 Plugin owner generation 已过期

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `registry` |
| 可重试   | 是         |

## 含义

PLG-3013 表示 Plugin owner generation 已过期。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

transaction、registration 或 cleanup 使用的 installation/generation 已不是当前 owner

## 建议操作

丢弃旧异步结果，并基于当前 generation 重新开始操作

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
