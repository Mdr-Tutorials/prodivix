# PLG-2004 Publisher 与插件 scope 不一致

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2004 表示 Publisher 与插件 scope 不一致。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

scoped plugin id 的 npm scope 与 `publisher` 不相同

## 建议操作

让 `publisher` 与插件 id 中的 scope 保持一致
