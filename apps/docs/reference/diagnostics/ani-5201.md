# ANI-5201 Effect capability 不受支持

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ANI       |
| 范围     | 动画      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

ANI-5201 表示 Effect capability 不受支持。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

effect host 未声明 track 所需的 style、css-filter 或 svg-filter capability

## 建议操作

切换 effect host 或调整 track

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
