# ANI-5202 Effect target 不可用

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ANI       |
| 范围     | 动画      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

ANI-5202 表示 Effect target 不可用。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

effect host 无法解析 document-qualified PIR target node

## 建议操作

修复动画 binding 或恢复目标节点

- 触发该错误的操作
- 可复现时的最小文档或配置
