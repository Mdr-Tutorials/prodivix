# PLG-4003 Runtime 操作超时

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 是        |

## 含义

PLG-4003 表示 Runtime 操作超时。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

activation 或 deactivation 超过 Host 配置的时间上限

## 建议操作

重试操作，或禁用持续无响应的插件

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
