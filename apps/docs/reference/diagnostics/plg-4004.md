# PLG-4004 Owner cleanup 不完整

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `cleanup` |
| 可重试   | 是        |

## 含义

PLG-4004 表示 Owner cleanup 不完整。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

runtime deactivation、subscription dispose、transaction rollback 或 contribution owner cleanup 失败

## 建议操作

重试 cleanup，并在移除或重新启用插件前确认 lease 已清零
