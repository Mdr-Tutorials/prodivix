# PLG-4025 Protocol request 超时

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `protocol` |
| 可重试   | 是         |

## 含义

PLG-4025 表示 Protocol request 超时。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

request 在 protocol deadline 内没有收到受验证 response

## 建议操作

仅在当前 runtime session 仍 active 时重试
