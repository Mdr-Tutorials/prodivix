# AI-5006 AI Apply token 缺失或过期

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | AI        |
| 范围     | AI 助手   |
| 严重程度 | `error`   |
| 阶段     | `command` |
| 可重试   | 是        |

## 含义

AI-5006 表示 AI Apply token 缺失或过期。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

用户尝试应用 AI patch，但 dry-run 生成的 apply token 不存在、已过期或对应 workspace revision 已变化

## 建议操作

重新 dry-run 后再应用

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
