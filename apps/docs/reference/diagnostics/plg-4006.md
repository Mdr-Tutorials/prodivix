# PLG-4006 Host operation 已被替代

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `info`    |
| 阶段     | `runtime` |
| 可重试   | 是        |

## 含义

PLG-4006 表示 Host operation 已被替代。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

disable、撤权、新 generation 或更新 operation supersede 仍在执行的旧异步操作

## 建议操作

使用最新 Plugin Host snapshot 决定是否重新发起操作

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
