# PLG-1015 Contribution 资源超出上限

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `parse` |
| 可重试   | 否      |

## 含义

PLG-1015 表示 Contribution 资源超出上限。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

单资源 bytes、单插件资源数、总 bytes、descriptor depth 或 node count 超过 Host 限制

## 建议操作

拆分或缩小 descriptor，并移除不必要的资源数据

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
