# PLG-4039 Gateway request 被取消

## 含义

PLG-4039 表示 Gateway request 被取消。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

caller、Host shutdown、disable 或 session disposal 在完成前取消调用

## 建议操作

仅从 active session 且确认旧 effect 未完成时重试
