# AI-5003 AI Action 编辑字段未授权

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | AI        |
| 范围     | AI 助手   |
| 严重程度 | `error`   |
| 阶段     | `command` |
| 可重试   | 否        |

## 含义

AI-5003 表示 AI Action 编辑字段未授权。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

AI 输出尝试修改 action capability 未允许的字段或 operation

## 建议操作

使用更具体的 AI 操作，或手动编辑该字段

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
