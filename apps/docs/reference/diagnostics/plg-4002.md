# PLG-4002 Runtime activation 失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 是        |

## 含义

PLG-4002 表示 Runtime activation 失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

runtime adapter 返回失败、抛错，或 Host 无法建立 termination listener

## 建议操作

检查 runtime diagnostics，修复后使用显式 retry
