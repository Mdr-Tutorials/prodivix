# PLG-4023 Protocol correlation 非法

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `protocol` |
| 可重试   | 否         |

## 含义

PLG-4023 表示 Protocol correlation 非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

response 缺少有效 replyTo、方向不匹配、重复回复或引用未知 pending request

## 建议操作

每个 request 只使用绑定 message id 回复一次
