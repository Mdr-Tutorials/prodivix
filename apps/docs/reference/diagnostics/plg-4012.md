# PLG-4012 Runtime artifact 超出上限

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

PLG-4012 表示 Runtime artifact 超出上限。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

runtime entrypoint bytes 超过 Host runtime artifact limit，即使 package reader 忽略了传入限额

## 建议操作

将 runtime 构建为低于 Host 上限的 self-contained ESM entry
