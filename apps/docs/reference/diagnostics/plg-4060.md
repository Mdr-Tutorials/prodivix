# PLG-4060 Required Gateway audit 不可用

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `audit` |
| 可重试   | 是      |

## 含义

PLG-4060 表示 Required Gateway audit 不可用。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

required-before-effect contract 没有 audit store，或 preflight record 未能持久提交

## 建议操作

恢复 Host 持久审计存储后重试敏感操作
