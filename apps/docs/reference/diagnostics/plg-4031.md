# PLG-4031 Gateway capability 当前被拒绝

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | PLG          |
| 范围     | 插件         |
| 严重程度 | `error`      |
| 阶段     | `permission` |
| 可重试   | 否           |

## 含义

PLG-4031 表示 Gateway capability 当前被拒绝。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

live permission snapshot 未 grant capability，或调用中途撤权

## 建议操作

通过 Host 权限流程授权后，从当前 session 重新发起调用

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
