# PLG-2015 资源路径不可移植或发生冲突

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2015 表示 资源路径不可移植或发生冲突。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

路径包含 Windows 保留设备名、尾随点空格，或多个路径在大小写不敏感文件系统上冲突

## 建议操作

使用唯一、规范且以 `./` 开头的包内相对路径

- 当前项目或工作区 ID

- 可复现时的最小文档或配置
