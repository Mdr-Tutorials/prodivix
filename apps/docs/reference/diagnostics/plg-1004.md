# PLG-1004 Manifest 不符合 v1 Schema

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PLG      |
| 范围     | 插件     |
| 严重程度 | `error`  |
| 阶段     | `schema` |
| 可重试   | 否       |

## 含义

PLG-1004 表示 Manifest 不符合 v1 Schema。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Manifest 字段、类型、枚举、格式、必填项或封闭对象规则不符合 Plugin Manifest v1 Schema

## 建议操作

按诊断中的 Manifest path 修正对应字段

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
