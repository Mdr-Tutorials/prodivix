# PLG-4015 Sandbox policy 无效

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `sandbox` |
| 可重试   | 否        |

## 含义

PLG-4015 表示 Sandbox policy 无效。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

iframe sandbox token、专用 origin、CSP、Permissions Policy 或 credential policy 不满足 Host 要求

## 建议操作

使用无 cookie 的专用 origin 和规定的 production policy
