# PLG-4020 Protocol message 非法

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |

## 含义

PLG-4020 表示 Protocol message 非法。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

MessagePort 收到非 string、非严格 JSON、超限或不符合 Runtime Envelope v1 的消息

## 建议操作

发送受限 strict JSON，并满足 exact envelope 和 payload contract
