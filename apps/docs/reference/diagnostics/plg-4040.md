# PLG-4040 Sandbox message quota 超限

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `quota` |
| 可重试   | 否      |

## 含义

PLG-4040 表示 Sandbox message quota 超限。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

runtime 超出单条 message、消息速率或 pending request 上限

## 建议操作

降低 runtime 流量并重启被终止的 session

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
