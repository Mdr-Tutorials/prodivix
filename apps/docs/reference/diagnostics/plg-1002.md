# PLG-1002 Manifest 包含重复对象键

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `parse` |
| 可重试   | 否      |

## 含义

PLG-1002 表示 Manifest 包含重复对象键。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

同一 JSON object 中出现两个或更多同名 property

## 建议操作

每个 object property 只保留一个明确值

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
