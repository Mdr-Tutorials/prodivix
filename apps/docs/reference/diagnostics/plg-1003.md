# PLG-1003 程序化输入不是 JSON value

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PLG      |
| 范围     | 插件     |
| 严重程度 | `error`  |
| 阶段     | `schema` |
| 可重试   | 否       |

## 含义

PLG-1003 表示 程序化输入不是 JSON value。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

程序化 Manifest 含 `undefined`、函数、symbol、BigInt、非有限 number、cycle、非普通对象、getter 或稀疏数组

## 建议操作

仅传入 JSON primitive、普通 data object 和稠密 array
