# PLG-4013 Sandbox bootstrap 失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `sandbox` |
| 可重试   | 是        |

## 含义

PLG-4013 表示 Sandbox bootstrap 失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

专用 sandbox origin、broker frame、nonce/source 校验或 bootstrap transfer 无法建立

## 建议操作

恢复专用 sandbox 部署与安全 headers 后重试 activation
