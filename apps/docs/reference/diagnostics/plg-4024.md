# PLG-4024 Protocol response 已迟到

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `warning`  |
| 阶段     | `protocol` |
| 可重试   | 否         |

## 含义

PLG-4024 表示 Protocol response 已迟到。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

response 在 request timeout、cancel 或 close 后到达

## 建议操作

丢弃迟到结果，并修复 runtime 的取消传播

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
