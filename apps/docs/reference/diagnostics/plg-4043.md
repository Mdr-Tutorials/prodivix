# PLG-4043 Gateway quota 超限

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `quota` |
| 可重试   | 否      |

## 含义

PLG-4043 表示 Gateway quota 超限。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

单 session 或 method 的 request rate/concurrency 超出 Host policy

## 建议操作

降低 Gateway 调用频率或等待当前请求结束

- 错误码和 requestId

- 触发该错误的操作
- 可复现时的最小文档或配置
