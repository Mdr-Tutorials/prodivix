# PLG-2014 Activation 缺少 runtime entrypoint

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2014 表示 Activation 缺少 runtime entrypoint。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Manifest 声明一个或多个 activation event，但没有 `entrypoints.runtime`

## 建议操作

声明 runtime module，或删除不需要的 activation event
