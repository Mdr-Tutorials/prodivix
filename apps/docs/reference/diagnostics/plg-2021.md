# PLG-2021 Contribution library owner 不一致

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2021 表示 Contribution library owner 不一致。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Render Policy、Codegen Policy、Icon Provider 或外部 Palette group 引用了不属于当前插件 owner 的 External Library

## 建议操作

在同一插件 package 中声明被引用的 External Library，或移除跨 owner 引用

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
