# PLG-1001 Manifest 源不是严格 UTF-8 JSON

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `parse` |
| 可重试   | 否      |

## 含义

PLG-1001 表示 Manifest 源不是严格 UTF-8 JSON。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Manifest 含无效 UTF-8、BOM、注释、尾逗号、语法错误或无法安全解析的 JSON 文本

## 建议操作

使用无 BOM 的 UTF-8 编码，并移除注释、尾逗号和非标准 JSON 语法
