# PLG-4030 Gateway capability 未在 Manifest 请求

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | PLG          |
| 范围     | 插件         |
| 严重程度 | `error`      |
| 阶段     | `permission` |
| 可重试   | 否           |

## 含义

PLG-4030 表示 Gateway capability 未在 Manifest 请求。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Gateway contract 推导出的 exact capability id/scope 不在 Plugin Manifest 中

## 建议操作

在 Manifest 声明该 exact capability 和 reason 后重新安装
