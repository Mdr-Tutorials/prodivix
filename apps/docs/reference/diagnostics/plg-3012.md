# PLG-3012 Contribution resolver 失败

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `registry` |
| 可重试   | 否         |

## 含义

PLG-3012 表示 Contribution resolver 失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Host-side resolver 抛错、产生非法 lifetime/order，或依赖未请求或未授权 capability

## 建议操作

修复 Host contribution adapter 或 descriptor 后重新验证插件

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
