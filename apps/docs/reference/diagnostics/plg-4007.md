# PLG-4007 Audit sink 不可用

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `warning` |
| 阶段     | `audit`   |
| 可重试   | 是        |

## 含义

PLG-4007 表示 Audit sink 不可用。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

audit event 创建失败，或 best-effort audit sink 抛错/拒绝 lifecycle event batch

## 建议操作

恢复 audit sink；在恢复前不要依赖完整的插件生命周期审计记录
