# PLG-3002 Capability policy 解析失败

## 快速信息

| 名称     | 说明         |
| -------- | ------------ |
| 前缀     | PLG          |
| 范围     | 插件         |
| 严重程度 | `error`      |
| 阶段     | `permission` |
| 可重试   | 是           |

## 含义

PLG-3002 表示 Capability policy 解析失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

policy adapter 抛错、返回错误 owner/revision、漏判请求、改变 optional 语义或尝试授权未请求 capability

## 建议操作

恢复权限策略来源并重新解析授权
