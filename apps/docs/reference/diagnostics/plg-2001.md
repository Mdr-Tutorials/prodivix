# PLG-2001 插件版本不是有效 SemVer

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2001 表示 插件版本不是有效 SemVer。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`version` 通过基础字符串格式后仍无法被严格 SemVer parser 接受

## 建议操作

使用完整且规范的 SemVer 版本，例如 `1.2.3`

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
