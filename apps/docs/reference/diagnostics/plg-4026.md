# PLG-4026 Protocol session 已关闭

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `protocol` |
| 可重试   | 是         |

## 含义

PLG-4026 表示 Protocol session 已关闭。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

request/event 在 endpoint close 后发送，或 pending request 因 close 被取消

## 建议操作

重启 runtime session 后再发送消息

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
