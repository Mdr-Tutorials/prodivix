# PLG-2013 Activation 引用无效

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2013 表示 Activation 引用无效。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

`contribution.use` 未引用同 point 的已声明 contribution，或宿主提供的 command catalog 中不存在 command id

## 建议操作

修正 contribution point、contribution id 或 command id

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
