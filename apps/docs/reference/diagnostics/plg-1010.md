# PLG-1010 Contribution 资源读取失败

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `parse` |
| 可重试   | 是      |

## 含义

PLG-1010 表示 Contribution 资源读取失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

package reader 无法在受限 package root 内读取 contribution resource

## 建议操作

检查资源路径、包内容和安装源可用性后重试

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
