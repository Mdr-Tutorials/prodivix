# ANI-5102 Easing 不受当前 Runtime 支持

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ANI       |
| 范围     | 动画      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

ANI-5102 表示 Easing 不受当前 Runtime 支持。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

timeline 或 keyframe easing 无法由当前 deterministic evaluator 执行

## 建议操作

使用受支持的 easing 或接入 custom easing CodeSlot runtime
