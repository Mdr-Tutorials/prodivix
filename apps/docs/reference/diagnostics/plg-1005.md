# PLG-1005 Manifest 超出资源上限

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `parse` |
| 可重试   | 否      |

## 含义

PLG-1005 表示 Manifest 超出资源上限。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Manifest byte size、JSON depth 或 JSON node count 超过宿主配置上限

## 建议操作

缩小 Manifest，并把大型 contribution descriptor 移到独立资源文件

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
