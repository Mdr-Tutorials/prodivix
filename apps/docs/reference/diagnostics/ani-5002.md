# ANI-5002 执行目标时间线不存在

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | ANI       |
| 范围     | 动画      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

ANI-5002 表示 执行目标时间线不存在。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

revision-bound Animation Job 指向的 timeline 不存在

## 建议操作

重新选择时间线并基于最新 Workspace revision 运行
