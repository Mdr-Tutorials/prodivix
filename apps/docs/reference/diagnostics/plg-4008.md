# PLG-4008 Host subscriber 回调失败

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `warning`  |
| 阶段     | `registry` |
| 可重试   | 否         |

## 含义

PLG-4008 表示 Host subscriber 回调失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Plugin Host 或 contribution registry subscriber 在处理已提交 snapshot/batch 时抛错

## 建议操作

修复或移除失败的宿主 subscriber

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
