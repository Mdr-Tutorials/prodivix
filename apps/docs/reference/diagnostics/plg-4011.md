# PLG-4011 Runtime artifact 完整性不匹配

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 否        |

## 含义

PLG-4011 表示 Runtime artifact 完整性不匹配。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Host 无法计算 runtime artifact SHA-256，digest 格式非法，或实际 digest 与 Manifest 声明的 integrity 不一致

## 建议操作

恢复与 Manifest/package attestation 匹配的 runtime artifact
