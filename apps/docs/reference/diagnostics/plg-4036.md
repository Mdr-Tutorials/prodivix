# PLG-4036 Gateway session 已过期

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `gateway` |
| 可重试   | 否        |

## 含义

PLG-4036 表示 Gateway session 已过期。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

owner、installation、generation、plugin version 或 bound session 已不再 current

## 建议操作

从当前 plugin generation 重新获取 session

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
