# PLG-4061 Gateway outcome audit 写入失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `warning` |
| 阶段     | `audit`   |
| 可重试   | 是        |

## 含义

PLG-4061 表示 Gateway outcome audit 写入失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

best-effort preflight 或 outcome record 未能在有界时间内持久化

## 建议操作

恢复 IndexedDB/audit backend，并检查 retention policy

- 触发该错误的操作
- 可复现时的最小文档或配置
