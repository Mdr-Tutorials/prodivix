# PLG-4010 Runtime artifact 读取失败

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `runtime` |
| 可重试   | 是        |

## 含义

PLG-4010 表示 Runtime artifact 读取失败。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

Host 无法从当前 installation package source 读取 Manifest 指定的 runtime entrypoint

## 建议操作

恢复已验证插件包中的 runtime artifact 后重试 activation
