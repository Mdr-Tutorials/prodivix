# PLG-1014 Contribution descriptor 不符合 contract

## 快速信息

| 名称     | 说明     |
| -------- | -------- |
| 前缀     | PLG      |
| 范围     | 插件     |
| 严重程度 | `error`  |
| 阶段     | `schema` |
| 可重试   | 否       |

## 含义

PLG-1014 表示 Contribution descriptor 不符合 contract。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

inline 或 resource descriptor 未通过 point-specific contract validator

## 建议操作

按 contribution contract 修正 descriptor 字段和值

- 错误码和 requestId

- 触发该错误的操作
- 可复现时的最小文档或配置
