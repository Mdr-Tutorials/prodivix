# PLG-2012 Contribution 缺少注册能力

## 快速信息

| 名称     | 说明       |
| -------- | ---------- |
| 前缀     | PLG        |
| 范围     | 插件       |
| 严重程度 | `error`    |
| 阶段     | `semantic` |
| 可重试   | 否         |

## 含义

PLG-2012 表示 Contribution 缺少注册能力。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

contribution 没有对应的 `extension.register` capability，或 capability scope 与 contribution point 不一致

## 建议操作

为该 contribution point 请求对应注册能力并说明 reason

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
