# PLG-1011 Contribution 资源不是严格 JSON

## 快速信息

| 名称     | 说明    |
| -------- | ------- |
| 前缀     | PLG     |
| 范围     | 插件    |
| 严重程度 | `error` |
| 阶段     | `parse` |
| 可重试   | 否      |

## 含义

PLG-1011 表示 Contribution 资源不是严格 JSON。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

contribution resource 含 BOM、无效 UTF-8、注释、尾逗号、重复键或其他非严格 JSON 内容

## 建议操作

将 descriptor 改为无 BOM 的严格 UTF-8 JSON，并移除重复对象键

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
