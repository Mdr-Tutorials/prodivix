# PLG-4042 Sandbox 已终止

## 快速信息

| 名称     | 说明      |
| -------- | --------- |
| 前缀     | PLG       |
| 范围     | 插件      |
| 严重程度 | `error`   |
| 阶段     | `sandbox` |
| 可重试   | 是        |

## 含义

PLG-4042 表示 Sandbox 已终止。请先确认当前页面、项目状态和最近操作，再按建议操作处理。

## 触发条件

broker、Worker、protocol violation、crash 或 Host 命令终止当前 sandbox

## 建议操作

检查稳定 reason code 后决定是否重启

## 上报时提供

- 错误码和 requestId
- 当前项目或工作区 ID
- 触发该错误的操作
- 可复现时的最小文档或配置
